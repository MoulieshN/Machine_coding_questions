This repository contains my solutions to various machine coding / low-level design (LLD) questions.
The goal of this repo is to practice writing clean, extensible, and production-ready code by applying object-oriented principles and design patterns.

## 🎯 Objectives

Improve problem-solving skills in machine coding rounds.

Write modular and testable code with clear abstractions.

Practice popular design patterns used in real-world systems.

Build a reference library of LLD implementations for future revision.

## 🛠️ Tech Stack

Language: Java (primary)

Core Concepts: OOPs, SOLID Principles, Design Patterns

Build Tool: Maven / Gradle (if applicable)
